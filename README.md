# Bean Battles

Bean Battles is a collaborative, Mario Party-inspired collection of short competitive and cooperative minigames. Players gather beans, face unpredictable events, and compete for the win across a shared game board.

## Project goals

- Make it easy for contributors to create and share self-contained minigames.
- Keep the core board game experience lightweight, approachable, and fun in local or online play.
- Build clear, reusable interfaces so minigames can plug into the main game without rewriting shared systems.

## Getting started

This repository is currently in its planning phase. The first implementation decisions will cover the game engine, supported platforms, core game loop, and the minigame extension interface.

## Contributing

Ideas, minigame concepts, art, music, code, and playtesting feedback are all welcome. Contribution guidelines and the technical architecture will be added as the project takes shape.
