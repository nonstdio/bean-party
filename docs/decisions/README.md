# Decision records

Create a decision record here for a consequential choice that would be expensive to reverse. Use a zero-padded number and a short slug, for example `0001-engine-and-first-platform.md`.

```md
# Decision: <title>

Date: YYYY-MM-DD
Status: Proposed | Accepted | Superseded

## Context

What problem are we solving? What constraints and assumptions matter?

## Options considered

- Option A — benefits and trade-offs
- Option B — benefits and trade-offs

## Decision

What did we choose, and why?

## Consequences

What becomes easier, harder, required, or explicitly out of scope?
```

Examples include the engine and version, first target platform, online-play milestone, project licenses, the shared minigame API, and the common art pipeline.
