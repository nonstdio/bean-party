# Keepaway Yard

Graybox minigame prototype testing whether simple bean movement and bumping produce readable, funny, recoverable competition.

## Players

2–4 local players on one shared screen.

## Objective

Hold the glowing objective to earn points every second. Touch it while loose to pick it up. Bump the holder to knock it loose. Highest score when the ~55 second timer ends wins.

## Controls

| Player | Move | Bump |
| --- | --- | --- |
| P1 | WASD | Space |
| P2 | Arrow keys | Enter |
| P3 | IJKL | U |
| P4 | Numpad 8456 | Numpad 0 |

Press Space or Enter on the briefing / ready screens to advance. `R` restarts after results. `Esc` exits to the starter app.

## Assets

Procedural Godot primitives only. No imported art, audio, or fonts.
